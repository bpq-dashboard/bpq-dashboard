Known Harmless Warnings — Don't Panic
WARNING Log file write error: [WinError 5] Access is denied: ...wavenode-log.tmp -> wavenode-log.json
Source: wavenode-reader.log on Windows. When: every 5 min during Linux SFTP sync. Cause: SFTP read handle on Windows blocks reader's atomic rename (~2-3 sec window). Impact: none — reader retries and succeeds. Permanent fix: NDJSON refactor (deferred, item #18 in pending list).
cosmetic
PHP Warning: rename(/var/www/bpq-dashboard/cache/syslog-parsed.json.tmp, ...): No such file or directory in network-api.php on line 319
Source: /var/log/nginx/error.log. When: two concurrent network_history requests overlap. Cause: shared tempfile path; one request renames it before the other gets there. Impact: none — both requests get valid data. Permanent fix: unique tempfile names per-request (item #9 in pending list, 5 min change).
cosmetic
access forbidden by rule, client: x.x.x.x, request: ".env" or ".git/config"
Source: /var/log/nginx/error.log. When: constantly, 24/7. Cause: internet-facing scanner bots probing for misconfigured apps. Impact: none — nginx returns 444 (silent close). Action: ignore. fail2ban will ban repeat offenders.
expected
